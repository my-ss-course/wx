## 小程序简介
小程序是一种全新的连接用户与服务的方式，它可以在微信内被便捷地获取和传播，同时具有出色的使用体验。

## 小程序代码构成
* .json 后缀的 JSON 配置文件
* .wxml 后缀的 WXML 模板文件
* .wxss 后缀的 WXSS 样式文件
* .js 后缀的 JS 脚本逻辑文件

## 调试
### vConsole
在真机上，如果想要查看 console API 输出的日志内容和额外的调试信息，需要在点击屏幕右上角的按钮打开的菜单里选择「打开调试」。此时小程序/小游戏会退出，重新打开后会右下角会出现一个 vConsole 按钮。点击 vConsole 按钮可以打开日志面板。
